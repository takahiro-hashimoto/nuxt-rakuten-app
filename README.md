# Nuxt.jsで楽天商品検索アプリを作ってみた

## 目的
* Nuxt.jsを触ってみる
* Nuxt.jsで作成したアプリをネットに公開するまでのフローを掴む
* Vuexの理解を深める

## 使用技術
* Nuxt.js（Vue.js）
* ElementUI（Vue.jsのコンポーネントライブラリ）
* Netlify（ホスティングサービス）
* 楽天API

## Nuxt.jsを使ってみた感想
* 開発環境を準備せずにすぐ制作に入れるのが便利
* Vue.jsの理解が浅いと辛い
* cssはどこまでを共通のcssに持たせるといいのか、どこまでをscopedで区切るといいかの線引きがわからない
